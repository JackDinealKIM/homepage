학원/교육기관 홈페이지 제작 프롬프트
프로젝트 개요
교육기관을 위한 반응형 웹사이트를 제작해주세요.
기술 스택

HTML5
jQuery
Tailwind CSS
ScrollReveal.js (스크롤 애니메이션)
Anime.js (인터랙티브 애니메이션)

디자인 컨셉 & 톤앤매너

컬러 팔레트:

주색상: 따뜻한 오렌지/브라운 계열 (#D97706, #92400E)
강조색: 빨간색 (#DC2626)
배경색: 크림/베이지 톤 (#FEF3C7)
텍스트: 다크 브라운 (#451A03), 화이트 (#FFFFFF)


분위기: 친근하고 따뜻한 교육기관, 신뢰감 있는 전문성
레이아웃: 모던하면서도 접근하기 쉬운 구조
타이포그래피: 가독성 좋은 한글 폰트 (Noto Sans KR 권장)

메뉴 구조
상위 메뉴 (헤더 네비게이션)

메인 - 홈페이지 메인
인사말/학원소개 - 원장 인사말, 학원 소개
수강후기 - 학생/학부모 후기
자주 묻는 질문 - FAQ 섹션
예약하기 - 상담 예약/문의

하위 메뉴 (드롭다운 또는 서브 네비게이션)

강사진 소개 - 선생님 프로필
프로그램/커리큘럼 - 수업 과정 안내
교재 안내 - 사용 교재 소개
오시는 길 - 위치 및 교통편

주요 기능 요구사항
1. 메인 페이지

히어로 섹션: 메인 슬로건과 CTA 버튼
주요 프로그램 소개 (카드 형태)
수강후기 미리보기
공지사항/소식 섹션

2. 애니메이션 요구사항

ScrollReveal: 스크롤 시 요소들이 부드럽게 나타나는 효과
Anime.js:

메인 타이틀 텍스트 애니메이션
버튼 호버 효과
카드 hover 인터랙션
로딩 애니메이션



3. 반응형 디자인

데스크톱 (1200px+)
태블릿 (768px-1199px)
모바일 (767px 이하)

4. 특별 기능

동영상/이미지 지원: 메인 페이지에 소개 동영상 또는 이미지 갤러리
팝업 모달: 공지사항, 이벤트 안내용
부드러운 스크롤: 앵커 링크 클릭 시 부드러운 이동

페이지별 상세 요구사항
메인 페이지
- Header: 로고, 네비게이션, 연락처
- Hero Section: 메인 비주얼, 핵심 메시지, CTA
- About Section: 학원 간단 소개
- Programs Section: 주요 프로그램 3-4개 (카드 형태)
- Testimonials: 수강후기 슬라이더
- News Section: 공지사항/소식
- Footer: 연락처, 주소, 소셜미디어
  인사말/학원소개 페이지

원장 인사말 (사진 포함)
학원 연혁
교육 철학
시설 소개 (이미지 갤러리)

프로그램/커리큘럼 페이지

프로그램별 상세 설명
레벨별 과정 안내
수업 시간표
수강료 안내

스타일링 가이드라인
컴포넌트 스타일
css- 버튼: 둥근 모서리, 그라데이션 효과, 호버 애니메이션
- 카드: 그림자 효과, 호버 시 살짝 올라오는 효과
- 폼: 부드러운 인풋 필드, 포커스 효과
- 네비게이션: 고정 헤더, 스크롤 시 배경 변경
  애니메이션 가이드

페이지 로딩: 0.5초 이내
스크롤 애니메이션: 부드럽고 자연스러운 타이밍
호버 효과: 즉각적인 반응 (0.2초 전환)
